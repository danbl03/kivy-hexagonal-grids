#### 개념

* 육각형(hexagon): 6면 다각형(6-sided polygons)
* 표준 육각형(regula hexagon): 모든 면의 모서리(edge) 길이가 동일한 육각형
* 대표 방향(typical orientation) 
    * 뽀족한 꼭대기(pointy topped): 수평(horizontal) 배치
    * 평평한 꼭대기(flat topped): 수직(vertical) 배치


#### 모서리(edges)

* 6 개 모서리 구성
* 2 개 육각형 1 개 모서리 공유


#### 꼭지점(corners)

* 6 개 꼭지점 구성
* 3 개 육각형 1개 꼭지점 공유
