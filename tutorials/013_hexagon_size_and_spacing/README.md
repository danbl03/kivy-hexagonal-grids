가로 정렬
---------
<p><img src="./screenshots/result_pointy_topped.png" width=50%"/></p>

* 높이: 모서리 길이 * 2
* 너비: 높이 * sqrt(3) / 2 = 모서리 길이 * sqrt(3) * 2
* 세로 간격: 높이 * 3 / 4 = 모서리 길이 * 3 / 2
* 가로 간격: 너비

---------
<p><img src="./screenshots/result_flat_topped.png" width=50%"/></p>
